### 🚨 **1. 검색 유입이 100% 즉시 이탈 — 봇 트래픽 의심**

**현상**: Organic Search 세션 9건(전일 대비 추정 +7~8건)이 모두 참여율 0%, 체류시간 0초를 기록. Google 4세션, Bing 4세션 모두 동일한 패턴.

**왜 이런 일이 생겼나**: 교차 지표 분석 결과 봇 트래픽 가능성 높음. ①avgSessionDuration 0~0.3초 ②pageViewsPerSession = 1 ③engagementRate = 0% ④GSC 검색 노출 데이터 0건으로 실제 Google 검색 유입과 불일치. 특히 Bing 4세션이 체류시간 0초인 것은 비정상적.

**비즈니스 임팩트**: 실제 검색 유입은 거의 없다는 뜻. SEO 효과가 전혀 작동하지 않고 있으며, 데모 요청으로 이어질 잠재고객 유입이 차단된 상태.

**→ 액션**: ①GA4에서 해당 검색 세션들의 user-agent 상세 분석해서 봇 확인 ②봇이라면 GA4 IP 필터링 설정으로 노이즈 제거 ③GSC에서 실제 검색 노출이 있는지 일주일 단위로 재확인

**비즈니스 임팩트**: 유입량은 늘었지만 사이트 탐색으로 이어지지 않아 데모 요청 등 전환 가능성 극히 낮음. 인지도 상승 효과는 있을 수 있으나 리드 생성엔 기여 못함.

**→ 액션**: ①GA4에서 Direct 세션 중 landingPage가 '/' 아닌 특정 페이지로 들어온 비중 확인(다크 소셜 판별) ②메인 페이지 CTA 버튼 위치와 문구 개선해서 제품 페이지로 유도율 높이기
