## 6. 인사이트

### 1️⃣ 모바일 UX 완전 실패 상태 — 긴급 개선 필요

**현상**: 모바일 2세션 모두 참여율 0%, 체류시간 0초로 즉시 이탈. 데스크톱 대비 모바일 성과가 극단적으로 낮음. 7일 추이에서도 모바일 트래픽 비중이 지속적으로 10% 내외로 낮은 상태.

**왜 이런 일이 생겼나**: 모바일 최적화가 되지 않아 사용자가 페이지 로딩 후 바로 이탈. 특히 SAP/ERP 관련 B2B 사이트임에도 모바일 접근성이 전무한 상태. 모바일에서 접근하는 사용자는 간단한 정보 확인 목적일 가능성이 높은데, 이를 전혀 충족하지 못함.

**비즈니스 임팩트**: B2B 의사결정자들도 이동 중이나 외부에서 모바일로 솔루션을 검토하는 경우가 증가. 모바일 UX 실패는 잠재 고객의 첫인상을 완전히 망치고, 경쟁사 대비 전문성 부족으로 인식될 위험.

**→ 액션**: 1) 메인 페이지 모바일 반응형 긴급 점검 2) 핵심 CTA 버튼(데모 요청) 모바일에서 클릭 가능한지 테스트 3) 제품 소개 섹션 모바일 가독성 개선 4) 모바일 페이지 로딩 속도 측정 및 최적화

### 2️⃣ LG CNS 레퍼럴 트래픽의 높은 품질 — 파트너십 기회

**현상**: comm.lgcns.com에서 3세션 유입, 참여율 66.7%로 전체 채널 중 가장 높음. 직접 유입(28.6%)과 검색 유입(33.3%)보다 2배 높은 참여도.

**왜 이런 일이 생겼나**: LG CNS는 SAP 구축/컨설팅 전문 기업으로, 해당 사이트 방문자들이 실제 SAP 테스트 자동화에 관심이 높은 타겟 오디언스. 체류시간은 6초로 짧지만 참여율이 높다는 것은 빠르게 핵심 정보를 파악하고 관심을 보인 것으로 해석.

**비즈니스 임팩트**: LG CNS 관련 레퍼럴은 실제 SAP 도입 기업이나 SI 파트너와 연결될 가능성이 높음. 이는 데모 요청으로 직결될 수 있는 고품질 리드 소스.

**→ 액션**: 1) LG CNS 사이트에서 어떤 페이지/맥락에서 링크되었는지 추적 2) LG CNS와의 파트너십 또는 케이스 스터디 협업 검토 3) SAP SI 파트너사 대상 콘텐츠 제작 고려 4) 해당 레퍼럴 소스 지속 모니터링

### 3️⃣ 체류시간 급감과 방치 탭 가능성 — 콘텐츠 집중력 점검

**현상**: 평균 체류시간이 어제 56초에서 오늘 22초로 60% 급감. 하지만 참여율은 35.3%→37.5%로 소폭 개선되어 모순적 패턴.

**왜 이런 일이 생겼나**: 어제(4/28) 데이터에서 avgSessionDuration이 56초로 길었던 것이 실제로는 방치 탭이었을 가능성. 오늘은 실제 관심 있는 방문자들이 빠르게 정보를 파악하고 행동(참여)하는 패턴으로 전환. 데스크톱 체류시간 26초, 모바일 0초를 보면 실제 참여는 데스크톱에서만 발생.

**비즈니스 임팩트**: 방치 탭 제거된 상태에서 실제 참여율 개선은 긍정적 신호. 하지만 22초라는 체류시간으로는 제품 가치 전달이 부족할 수 있어 전환율에 한계.

**→ 액션**: 1) 메인 페이지 핵심 메시지를 15초 내 파악 가능하도록 구조 개선 2) 주요 차별화 포인트(50x 빠른 실행, No-code)를 상단 히어로 섹션에 명확히 배치 3) 스크롤 없이 보이는 영역에 데모 요청 CTA 강화

### 4️⃣ 검색 유입 개선 신호 — 하지만 GSC 데이터 부족으로 검증 불가

**현상**: Google organic 검색에서 3세션 유입, 참여율 33.3%, 체류시간 17초. 하지만 GSC 데이터가 null이어서 실제 검색 성과 검증 불가능.

**왜 이런 일이 생겼나**: GA4에서는 organic 검색 유입이 포착되지만 GSC 연동이나 데이터 수집에 문제가 있어 교차 검증이 불가. 검색 유입 1명이 3세션을 생성한 것으로 보아 동일 사용자의 반복 방문 가능성.

**비즈니스 임팩트**: SEO 성과를 정확히 측정할 수 없어 검색 마케팅 전략 수립에 차질. 검색은 가장 중요한 B2B 유입 채널 중 하나인데 성과 추적이 불가능한 상태.

**→ 액션**: 1) GSC 연동 상태 긴급 점검 및 재설정 2) Bing Webmaster Tools도 함께 확인하여 비Google 검색엔진 유입 파악 3) GA4에서 organic 검색 키워드 데이터 확인 가능한지 점검 4) 검색 콘솔 데이터 정상화 이후 주요 키워드 순위 모니터링 체계 구축

### 5️⃣ 랜딩 페이지 다양성 부족 — SEO 콘텐츠 확장 필요

**현상**: 전체 16세션 중 12세션(75%)이 메인 페이지로 유입. 나머지 4세션도 UTM 파라미터가 붙은 메인 페이지 변형으로, 실질적으로 메인 페이지 집중도가 100%에 가까움.

**왜 이런 일이 생겼나**: 블로그나 제품 상세 페이지로의 직접 유입이 전무. 이는 SEO 최적화된 콘텐츠가 부족하거나, 검색에서 발견되지 않고 있음을 의미. UTM 파라미터는 이메일 마케팅 캠페인에서 온 것으로 추정.

**비즈니스 임팩트**: 메인 페이지만으로는 다양한 구매 여정 단계의 잠재 고객을 포착하기 어려움. 특히 B2B에서는 문제 인식→솔루션 탐색→비교 검토 단계별로 다른 콘텐츠가 필요한데, 현재는 최종 의사결정 단계 방문자만 포착 중.

**→ 액션**: 1) SAP 테스트 관련 롱테일 키워드 타겟 블로그 콘텐츠 제작 2) /why-perfectwin, /solutions 페이지 SEO 최적화하여 검색 진입점 다양화 3) 제품 기능별 상세 페이지(/product/erp/test-execution 등)의 검색 노출 강화 4) "SAP 테스트 자동화", "ERP 테스트" 등 핵심 키워드 현재 순위 확인
