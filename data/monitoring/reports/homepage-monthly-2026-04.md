## 8. 인사이트

### 🔴 1. 트래픽 소스 의존도 위기 — Direct 채널 붕괴
**현상**: Direct 채널 세션이 1,579 → 532 (-66.3%)로 급락하면서 전체 트래픽의 80.2%를 차지하지만 절대량은 3분의 1로 축소. 3개월 추이를 보면 2월 7,582 → 3월 1,579 → 4월 532로 지속 하락세.

**왜 이런 일이 생겼나**: Direct 트래픽은 브랜드 검색, 북마크, 직접 URL 입력이 주요 소스인데, 다른 채널(Referral -72.3%, Organic Search -31.4%)도 동반 하락한 점으로 미뤄 브랜드 노출 자체가 감소했거나 마케팅 활동이 줄어든 것으로 추정. Teams 링크 유입도 27 → 6 (-77.8%)로 급감하여 내부 공유/추천도 줄어든 상황.

**비즈니스 임팩트**: 현재 트래픽의 80%가 Direct인데 이마저 월 1,000 세션 이상 감소하면 데모 요청이 현재 52건에서 20건 이하로 떨어질 위험. 브랜드 인지도 하락의 신호이며, 신규 고객 발견 경로가 막힌 상태.

**→ 액션**: 
- **즉시**: 지난 3월까지 진행했던 마케팅 활동 리스트업 후 중단된 것 재개 (우선순위 상)
- **2주 내**: LinkedIn/Twitter 포스팅 빈도 2배 증가로 브랜드 노출 복구
- **1개월 내**: 기존 고객/파트너 대상 레퍼럴 프로그램 런칭 (난이도 중)

### 🔴 2. 페이지 탐색 깊이 붕괴 — 1페이지 읽고 끝 패턴 고착화
**현상**: Pages/Session이 2.9 → 1.58 (-45.5%)로 급락. 바운스율도 55.39% → 68.55% (+13.16%p)로 악화. 체류시간은 182초 → 213초로 오히려 늘어났지만 실제 페이지 탐색은 절반으로 감소.

**왜 이런 일이 생겼나**: 체류시간 증가 + 페이지뷰 급감 + 바운스율 악화 조합은 전형적인 방치 탭 패턴. 사용자가 페이지를 열어놓고 실제로는 읽지 않거나, 첫 페이지에서 원하는 정보를 찾지 못해 추가 탐색을 포기하는 것으로 판단. Desktop 비중이 90%인 점도 방치 탭 가능성을 높임.

**비즈니스 임팩트**: 방문자가 제품 정보나 차별화 포인트를 충분히 탐색하지 않으면 전환 가능성이 낮아짐. 현재 메인 → 데모 요청 전환율이 12.1% → 7.8%로 하락한 것이 이를 뒷받침. 단순 방문이 아닌 '관심있는 방문'으로 전환되지 못하는 구조적 문제.

**→ 액션**:
- **즉시**: 메인 페이지 첫 화면에 "15초 제품 데모 영상" 추가로 즉시 관심 유발 (우선순위 최상)
- **1주 내**: 메인 페이지 하단에 "다음으로 읽어볼 만한 페이지" 추천 섹션 추가
- **2주 내**: 각 페이지 하단에 관련 페이지로의 명확한 CTA 버튼 배치 (난이도 하)

**현상**: Mobile 세션 비중이 10.8% → 9.9%로 소폭 감소했지만, Desktop(-65.1%) 대비 상대적으로 선방(-67.7%). Mobile에서의 참여도나 페이지 탐색 패턴은 별도 집계되지 않았으나, 절대 감소폭이 Desktop보다 작음.

**왜 이런 일이 생겼나**: B2B SaaS 특성상 Desktop 중심 사용이 일반적이지만, Mobile에서도 정보 탐색이나 초기 리서치가 발생. Mobile 트래픽이 상대적으로 덜 떨어진 것은 검색 행동이 모바일로 이동하는 트렌드 반영이거나, 특정 소셜 미디어 채널에서의 유입이 모바일 중심이기 때문일 수 있음.

**비즈니스 임팩트**: Mobile 최적화가 잘 되어 있다면 전체 트래픽 감소 상황에서도 모바일을 통한 리드 생성 기회를 확대할 수 있음. 특히 젊은 IT 의사결정자들은 모바일 리서치를 선호하는 경향이 있어 이 채널 강화가 신규 고객 발굴에 도움.

**→ 액션**:
- **1주 내**: Mobile 페이지 로딩 속도 및 UX 점검 (특히 데모 요청 페이지) (우선순위 중)

**현상**: 4월 GSC 데이터가 13일만 수집되었고 clicks, impressions가 모두 0으로 표시. 3월도 4일만 수집되어 검색 성과를 정확히 파악할 수 없는 상태. GA4 Organic Search는 71 세션이 있었으나 GSC와 대조 불가.

**왜 이런 일이 생겼나**: GSC 연동 설정 문제이거나 도메인 소유권 확인 이슈, 또는 실제로 Google 검색 결과에서의 노출이 극히 적을 가능성. GA4에서 Organic Search 71 세션이 집계된 점으로 미뤄 연동 문제일 가능성이 높음.

**비즈니스 임팩트**: SEO 성과를 측정할 수 없으면 검색 트래픽 개선 전략을 수립할 수 없음. 특히 "SAP 테스트 자동화", "S/4HANA 마이그레이션" 등 타겟 키워드에서의 순위나 노출량을 모르면 콘텐츠 최적화나 기술적 SEO 개선이 어려움.

**→ 액션**:
- **즉시**: GSC 연동 상태 점검 및 재설정 (perfectwin.ai 도메인 소유권 확인) (우선순위 최상)
- **2일 내**: GSC 연동 복구 후 지난 3개월 데이터 소급 확인
- **1주 내**: 주요 타겟 키워드에 대한 수동 검색 순위 체크 (난이도 하)

### 🟡 8. 요일별 패턴 최적화 기회 — 화요일 집중 현상
**현상**: 요일별 평균 세션을 보면 화요일(34.5)이 가장 높고, 월요일(15.8)이 상대적으로 낮음. 주말(일요일 8, 토요일 20)은 예상대로 낮지만, 평일 내에서도 편차가 큰 상황.

**왜 이런 일이 생겼나**: B2B 업무 패턴상 월요일은 주간 계획 수립으로 바쁘고, 화요일부터 본격적인 업무와 리서치가 시작되는 것으로 추정. SAP 테스트 자동화 같은 기술적 의사결정은 여유있게 검토할 수 있는 화-목요일에 집중되는 경향.

**비즈니스 임팩트**: 화요일에 트래픽이 집중된다는 것을 활용하여 해당 요일에 맞춘 콘텐츠 발행이나 캠페인 타이밍을 조정하면 더 높은 engagement를 얻을 수 있음. 반대로 월요일 저조함을 개선할 수도 있음.

**→ 액션**:
- **1주 내**: 블로그 포스팅을 화요일 오전에 발행하도록 스케줄 조정 (우선순위 하)
- **2주 내**: LinkedIn/Twitter 주요 콘텐츠를 화-목요일에 집중 배치
- **1개월 내**: 월요일 트래픽 개선을 위한 "주간 SAP 테스트 팁" 시리즈 런칭 고려 (난이도 하)
