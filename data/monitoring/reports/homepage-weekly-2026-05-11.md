## 6. 인사이트

### 1. Microsoft Teams 기반 추천 트래픽이 첫 데모 신청 견인
**현상**: MS Teams CDN(statics.teams.cdn.office.net)에서 4세션 유입, 이 중 1건이 데모 신청으로 전환. Referral 트래픽 중 36% 차지.

**원인**: Microsoft Teams에서 PerfecTwin 관련 문서나 링크가 공유된 것으로 추정. SAP 관련 업무 맥락에서 자연스럽게 노출.

**비즈니스 임팩트**: B2B SaaS의 이상적인 유입 경로. 업무용 메신저에서의 추천은 높은 구매 의향을 내포.

**→ 액션**: Teams에서 공유된 콘텐츠/맥락을 역추적하여 유사한 추천 기회 확장. SAP 커뮤니티나 업무 채널에서의 자연스러운 언급 유도 전략 수립.

### 2. 메인 페이지의 전환 유도력이 데모 페이지보다 우수
**현상**: 메인 페이지 랜딩 → 데모 신청 전환율 66.7% vs 데모 페이지 직접 랜딩 25%.

**원인**: 메인 페이지는 제품 가치 제안을 먼저 이해시킨 후 CTA로 유도하는 구조. 데모 페이지는 바로 폼 노출로 심리적 저항 발생.

**비즈니스 임팩트**: 메인 페이지 트래픽 증가 = 데모 신청 증가로 직결되는 구조.

**→ 액션**: 메인 페이지의 CTA 배치와 메시지를 데모 페이지에도 적용. 데모 페이지 상단에 간단한 가치 제안 섹션 추가 (폼 전에 "왜 PerfecTwin인가?" 요약).

### 3. 모바일 트래픽 급증했으나 참여도 검증 필요
**현상**: 모바일 세션이 5회→34회로 6배 증가. 전체 트래픽의 36%를 차지.

**원인**: Teams나 기타 모바일 메신저를 통한 링크 공유 추정. 하지만 모바일 engagementRate 별도 집계 없어 품질 미확인.

**비즈니스 임팩트**: B2B SaaS는 데스크톱 전환이 일반적이므로, 모바일 트래픽이 실제 비즈니스에 기여하는지 의문.

**→ 액션**: 다음 주 모바일/데스크톱별 engagementRate 분리 집계. 모바일 사용자의 데모 신청 경로 추적하여 모바일 UX 최적화 필요성 판단.

### 4. GSC 검색 데이터 완전 공백 상태
**현상**: Google Search Console에서 impressions 0, clicks 0. 2주 연속 검색 노출 전무.

**원인**: 
1) 신규 도메인으로 Google 인덱싱 미완료
2) SEO 기본 설정 누락 (sitemap, meta tags 등)
3) 콘텐츠 부족으로 검색 가치 없음

**비즈니스 임팩트**: Organic Search 21세션이 모두 Bing 등 타 검색엔진에서 유입. Google 검색 시장 점유율(90%+)을 고려하면 기회 손실 심각.

**→ 액션**: 
1) Google Search Console에서 사이트맵 제출 상태 확인
2) 메인 페이지 title, meta description 최적화
3) 핵심 키워드("SAP testing automation") 기반 기본 SEO 설정

### 5. 수요일 스파이크는 일회성, 지속성 있는 트래픽 확보 필요
**현상**: 수요일 43세션으로 집중, 다른 평일은 11-20세션 수준.

**원인**: 특정 공유나 이벤트로 인한 일회성 패턴. 요일별 일관성 없음.

**비즈니스 임팩트**: 불안정한 트래픽 패턴으로 예측 가능한 성장 어려움.

**→ 액션**: 정기적인 콘텐츠 발행 스케줄 수립. 매주 화/목요일 LinkedIn 포스팅으로 안정적인 B2B 트래픽 기반 구축.

### 6. Direct 트래픽 비중 과다, 채널 다각화 시급
**현상**: Direct 트래픽이 60세션(63.8%)으로 여전히 최대 비중.

**원인**: 브랜드 인지도 부족으로 다양한 채널에서의 자연 유입 부재.

**비즈니스 임팩트**: 특정 채널에 의존하지 않아 리스크는 낮지만, 확장성 제한.

**→ 액션**: LinkedIn 개인 브랜딩으로 Organic Social 비중 확대. SAP 관련 커뮤니티에서 가치 제공형 참여로 Referral 다각화.
