**1. 데모 퍼널 완전 중단 — 홈페이지 CTA 효과 실종**
- **현상**: 데모 페이지 도달 0건 (전주 1건에서 완전 중단). 홈페이지 세션은 54 (+58.8%)로 급증했지만 데모 전환 경로가 끊어짐
- **왜**: GA4 topPages에서 데모 페이지(`/contact-us/request-demo`) 방문은 2건 있었으나 demoFunnel에서는 0건 집계. 세션 정의 차이나 추적 설정 이슈 가능성
- **임팩트**: 트래픽 증가가 비즈니스 성과로 전환되지 않는 상황. 리드 생성 파이프라인 중단
- **→ 액션**: 홈페이지 `/contact-us/request-demo` 페이지의 GA4 이벤트 추적 설정 점검. 메인 페이지 CTA 버튼 위치·문구 테스트

**2. Bing 검색 유입 6건 모두 봇 트래픽 의심**
- **현상**: Bing organic 6세션 모두 체류시간 0초, 참여율 0%, 메인 페이지만 조회 후 이탈
- **왜**: 정상 사용자는 최소 1~2초는 체류하며 일부는 다른 페이지 탐색. 이 패턴은 자동화된 크롤링 봇 특징과 일치
- **임팩트**: 트래픽 수치는 부풀리지만 실제 사용자 인사이트 왜곡. 분석 정확도 하락
- **→ 액션**: GA4에서 Bing referrer 세션의 User-Agent 분석. 봇으로 확인되면 필터 설정으로 향후 집계 제외

**3. GSC vs inblog Organic 수치 불일치 — 집계 기준 차이**
- **현상**: GSC 총 클릭 1건, inblog Organic 18건 (18배 차이). GSC는 2일 지연이지만 이 격차는 정상 범위 초과
- **왜**: inblog는 referrer가 "google.com" 등인 모든 세션을 Organic으로 분류. 여기에는 Google Discover, News, 직접 링크 공유 등이 포함되어 실제 검색 클릭보다 높게 집계
- **임팩트**: 블로그 SEO 성과를 과대평가할 위험. 실제 검색 유입은 1건으로 매우 낮음
- **→ 액션**: 다음 주부터 블로그 리포트에서 "inblog Organic ≠ GSC 검색 클릭" 차이점을 매번 명시. SEO 성과 판단은 GSC 클릭 기준 사용

**4. 첫 페이지 노출 포스트 6개, 클릭 1건만 발생**
- **현상**: GSC에서 1~10위 노출된 포스트 6개 (SAP Cloud ALM Test Management 99노출·10위 등)에서 클릭은 "SAP Test Case Design" 1건뿐. CTR 0.15% (업계 평균 2~3% 대비 현저히 낮음)
- **왜**: 메타 디스크립션·제목이 검색 의도와 불일치하거나 경쟁 페이지 대비 클릭 유인 부족. 특히 "Test Management" "Upgrade Testing" 등은 기술 중심 제목으로 비즈니스 가치 어필 약함
- **임팩트**: 첫 페이지 진입 성과를 클릭으로 전환하지 못해 트래픽 획득 기회 손실. SEO 투자 ROI 낮음
- **→ 액션**: 상위 3개 포스트 (SAP Cloud ALM Test Management, S/4HANA Migration Testing, S/4HANA Upgrade Testing)의 메타 디스크립션을 "비즈니스 임팩트" 중심으로 리라이트
