### 1. **Teams 공유 → 데모 신청의 50% 직접 기여**
- **현상**: 데모 신청 2건 중 1건이 Microsoft Teams 공유에서 유입. 전환율 100%로 전일 20% 대비 80%p 급상승.
- **왜**: `statics.teams.cdn.office.net` 리퍼럴이 메인 페이지로 유입되어 바로 데모 신청. 기업 내부에서 PerfecTwin이 활발히 공유되고 있음.
- **임팩트**: B2B 특성상 기업 내 의사결정자 간 공유가 가장 강력한 전환 동력. 품질 높은 리드 창출.
- **→ 액션**: Teams/Slack 등 기업 메신저 공유에 최적화된 메인 페이지 메타 정보 점검. 공유 시 미리보기가 데모 신청 동기를 높이도록 제목/설명 개선.

### 2. **홈페이지 트래픽 58% 급감했지만 체류시간 증가는 방치 탭 착시**
- **현상**: 세션 20 (전일 43, -53.5%), 평균 체류시간 1분 31초 (+37초), 하지만 참여율 30% (-9.5%p).
- **왜**: 체류시간 증가 + 참여율 하락 + 세션당 페이지뷰 1.9는 방치 탭 패턴. Desktop 세션(15개)에서 평균 체류 1분 47초로 Mobile(44초)보다 2.4배 길어 방치 가능성 높음.
- **임팩트**: 실제 관심도는 떨어졌으나 수치상 "체류시간 좋다"로 오해할 수 있음.
- **→ 액션**: 체류시간 단독 지표 의존 중단. 참여율 + 세션당 페이지뷰 조합으로 실제 관심도 측정.

### 3. **한글 블로그 검색 유입 7배 우위, 영문 블로그는 검색에서 외면**
- **현상**: 한글 블로그 organic 7건 vs 영문 1건. GSC에서 블로그 노출 278건이지만 클릭 0건.

## 📚 용어 정리

**데모 퍼널**
- 도달 — 데모 폼 페이지를 본 세션 수. "관심" 신호.
- 완료 — thank-you 페이지 도달 = 영업 메일 발생.
- 전환율 — 도달자 중 완료 비율. 정상 5~10%.
- 첫 진입 페이지 (Landing Page) — 사용자가 가장 먼저 본 페이지. "어떤 콘텐츠가 데려왔는가"의 가장 강한 신호.

**트래픽 (GA4)**
- 방문자 (Active Users) — 그날 들른 고유 사람 수.
- 세션 (Sessions) — 방문 횟수. 30분 비활성 후 새 세션.
- 페이지뷰 (Page Views) — 페이지 본 총 횟수.
- 참여율 (Engagement Rate) — 10초+ 머물거나 2+ 페이지 본 세션 비율. **50%+ 좋음**.
- 세션당 페이지뷰 — 한 방문당 평균 페이지 수. **1.5+ 탐색 의향**.
- 평균 체류 시간 — 한 세션 평균 사이트 머문 시간. **2분+ 좋음**.

**검색 (GSC / Bing)**
- 노출 (Impressions) — 검색 결과에 우리가 보인 횟수.
- 클릭 (Clicks) — 실제 클릭 수.
- CTR (Click-Through Rate) — 클릭/노출. 평균 2~3%, **5%+ 좋음**.
- 평균 순위 (Position) — 검색 결과 우리 평균 위치. **10 이하 = 첫 페이지**, 11~20 = 둘째 페이지(클릭 급락), 20+ = 사실상 미노출.

**블로그 (inblog)**
- visits — 블로그 방문 횟수.
- post — 포스트 페이지를 본 횟수 (홈/카테고리 제외).
- clicks (CTA 클릭) — 블로그 본문 안 버튼·링크 클릭 수. CTA = Call To Action.
- organic — 검색엔진에서 자연 유입된 방문.

**어트리뷰션**
- Source — 어디서 왔는지 (google, linkedin.com 등).
- Medium — 어떻게 왔는지 (organic=자연검색, social=소셜, referral=다른 사이트 링크, direct=직접 입력).
