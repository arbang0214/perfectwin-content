## 5️⃣ 오늘의 종합 인사이트

### 1. 리퍼럴 트래픽의 질 문제
- **현상**: hanbijegarl.framer.website에서 2세션 유입 (전체의 28.6%)했지만 참여율(engagementRate) 50%, 평균 체류시간 7.8초로 매우 짧음
- **왜**: 이 소스는 사람이지만 콘텐츠 관심도가 낮거나, 잘못된 기대를 갖고 들어온 것. framer.website는 웹사이트 빌더이므로 개인 포트폴리오나 실험 사이트에서의 링크일 가능성
- **임팩트**: 트래픽 수치는 올리지만 데모 의향이나 브랜드 인지에는 기여 없음. 허수 트래픽
- **→ 액션**: 해당 리퍼럴 사이트 확인하여 어떤 맥락에서 링크가 걸렸는지 파악. 무관한 링크라면 향후 분석에서 제외 필터 적용

### 2. 블로그 검색 노출은 많지만 클릭 0
- **현상**: GSC에서 블로그 노출 203회 기록했으나 클릭은 0건. 평균 순위(position) 20.4는 두 번째 페이지 하단에 해당
- **왜**: 검색 결과 2페이지(11~20위)에서는 클릭율이 극히 낮음(보통 1% 미만). 메타 타이틀/디스크립션이 매력적이지 않거나, 검색 의도와 불일치할 수 있음
- **임팩트**: 브랜드 노출 효과는 있지만 실제 트래픽 증가로 이어지지 않음. SEO 투자 대비 성과 부족
- **→ 액션**: 상위 노출 페이지 중 순위 10위 내 진입 가능한 것(현재 9.2위인 "SAP Cloud ALM" 포스트 등) 위주로 메타 최적화 집중

### 3. 검색엔진별 성과 편차 심화
- **현상**: Google에서는 블로그 노출 203회 vs Bing 0회. 홈페이지도 Google/Bing 모두 노출 0
- **왜**: Bing SEO 최적화가 전혀 안 되어 있거나, 사이트맵/크롤링 설정에 문제 있을 수 있음. 또는 Bing의 인덱싱 지연
- **임팩트**: Microsoft 생태계(기업 환경) 사용자는 Bing을 쓰는 경우가 많은데, B2B SaaS 타겟을 놓치고 있을 가능성
- **→ 액션**: Bing Webmaster Tools에서 사이트맵 제출 확인, 크롤링 오류 점검. 최소한 홈페이지는 Bing에서도 검색되도록 설정

### 4. Direct 트래픽의 높은 이탈률 패턴
- **현상**: Direct 채널 2세션 모두 참여율 50%이지만 페이지/세션 1.0으로 다른 페이지 탐색 없음. 블로그 Direct도 21방문 모두 단일 포스트만 조회
- **왜**: 특정 페이지 URL을 직접 알고 온 사용자들이 해당 콘텐츠만 소비하고 사이트 전체 탐색으로 이어지지 않음. 내부 링크나 관련 컨텐츠 추천이 약한 것
- **임팩트**: 개별 콘텐츠는 소비되지만 브랜드 전체 이해나 데모 의향으로 발전하지 않음
- **→ 액션**: 블로그 포스트 하단에 관련 포스트나 솔루션 페이지로의 CTA 추가. 홈페이지 메인에서 주요 솔루션 페이지로의 네비게이션 개선
