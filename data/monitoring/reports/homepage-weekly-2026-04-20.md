## 6. 인사이트

### 🚨 1. 데모 전환율 급락 — 핵심 문제
**현상**: 데모 요청이 13건 → 4건 (-69.2%, -9건)으로 급감. CVR도 14.1% → 6.25% (-7.85%p) 하락.

**왜 이런 일이 생겼나**: 
- 메인 페이지 PV는 107 → 64 (-40.2%)로 감소했지만, 데모 페이지 PV는 -69.2%로 더 큰 폭 하락
- 메인에서 데모 페이지로의 내부 이동률이 크게 떨어짐 (12.1% → 6.25%)
- avgSessionDuration이 183초 → 82초 (-55%)로 급감해 사용자들이 충분히 탐색하지 않고 있음

**비즈니스 임팩트**: 주요 전환 목표인 데모 요청이 크게 감소해 잠재 고객 확보에 직접적 타격. 매주 이 수준이면 월간 리드 목표 달성 어려움.

**→ 액션**: 
- 메인 페이지의 데모 요청 CTA 위치/색상/문구를 더 눈에 띄게 개선
- 메인 페이지 로딩 속도 확인 (체류시간 급감 원인 파악)
- Why PerfecTwin → Demo 연결 동선 강화 (차별화 포인트 이해 후 전환 유도)

### 📈 2. Organic Search 비중 증가하나 절대량 부족
**현상**: Organic Search가 8세션 → 13세션 (+62.5%, +5세션)으로 증가. 전체 비중도 8.7% → 20.3%로 상승.

**왜 이런 일이 생겼나**:
- Bing organic이 2 → 4세션으로 2배 증가
- Google organic도 6 → 9세션으로 50% 증가
- GSC 데이터 부재로 구체적 키워드/순위 변화 추적 불가
- Direct 트래픽 감소로 상대적 비중이 높아진 효과도 있음

**비즈니스 임팩트**: SEO 개선 신호이나 절대량(13세션)이 여전히 낮아 브랜드 인지도 확산에는 제한적.

**→ 액션**:
- GSC 데이터 연동 복구하여 검색 성과 정확한 추적
- Bing Webmaster Tools 등록으로 Bing 검색 최적화 강화
- 현재 상승 중인 키워드 파악하여 관련 콘텐츠 확대

### ⚠️ 3. Direct 트래픽 의존도 심화와 방치 탭 의심
**현상**: Direct 세션이 75 → 47 (-37.3%)로 감소했지만 여전히 73.4% 비중. avgSessionDuration은 183초 → 82초로 급감.

**왜 이런 일이 생겼나**:
- engagementRate 31.58% + pageViewsPerSession 1.4 조합으로 볼 때 방치 탭 가능성
- desktop avgSessionDuration이 mobile보다 길 가능성 (데스크톱 87.5% 비중)
- 브랜드 직접 검색이나 북마크로 와서 메인만 보고 이탈하는 패턴

**비즈니스 임팩트**: 높은 Direct 의존도는 마케팅 채널 다각화 필요성을 시사하며, 방치 탭이 많다면 실제 관심도가 과대평가되고 있음.

**→ 액션**:
- 메인 페이지에서 다음 행동을 유도하는 명확한 내비게이션 개선
- 브랜드 인지 외 다른 유입 채널 확대 (LinkedIn, 콘텐츠 마케팅)
- 기기별 체류시간 분석으로 방치 탭 패턴 정확히 파악

### 💡 4. Solutions 페이지 관심도 증가 신호
**현상**: Solutions/Migration 페이지가 2 PV → 4 PV (+100%)로 증가. Solutions/Regression도 새로 3 PV 기록.

**왜 이런 일이 생겼나**:
- S/4HANA 마이그레이션 관련 검색 증가 가능성
- 메인 → Solutions 탐색 패턴이 늘어남
- 기존 제품 소개보다 솔루션 중심 접근에 관심 집중

**비즈니스 임팩트**: 구체적 업무 시나리오 중심의 솔루션 접근이 효과적임을 시사. 타겟 고객이 기능보다 "내 문제 해결" 관점으로 접근.

**→ 액션**:
- Solutions 페이지에서 Demo 요청으로의 전환 동선 강화
- Migration/Regression/Upgrades 각 솔루션별 구체적 성공 사례 추가
- 메인 페이지에서 Solutions 섹션을 더 눈에 띄는 위치로 이동

### 🌏 5. 인도 시장 신규 진입과 아시아 확산
**현상**: 인도에서 4세션 (6.3%) 신규 유입. 싱가포르도 2세션 기록.

**왜 이런 일이 생겼나**:
- SAP 구축이 활발한 인도 IT 기업들의 테스트 자동화 관심 증가
- 아시아 지역 SAP 컨설팅 펌이나 엔터프라이즈의 검색 유입 가능성
- 기존 한국(15세션), 미국(32세션) 외 지역 다변화

**비즈니스 임팩트**: 글로벌 확장 가능성을 보여주나, engagementRate나 전환율 추적 필요. 아시아 시장 특화 접근 기회.

**→ 액션**:
- 인도/싱가포르 유입의 engagementRate, 체류시간 별도 분석
- 아시아 타임존 고려한 데모 스케줄링 옵션 추가 검토
- SAP 아시아 지역 이벤트나 컨퍼런스 연계 마케팅 검토

### 📱 6. 모바일 트래픽 비중 증가하나 절대량 여전히 낮음
**현상**: 모바일 비중이 5.4% → 12.5%로 2배 이상 증가. 모바일 세션은 5 → 8 (+60%).

**왜 이런 일이 생겼나**:
- B2B SaaS 특성상 여전히 데스크톱 중심(87.5%)이나 모바일 검색 증가
- 모바일에서 pageViews/sessions = 1.0으로 단일 페이지 조회 후 이탈
- 모바일 UX 최적화 부족으로 추가 탐색 없이 바로 이탈

**비즈니스 임팩트**: 모바일 유입은 늘었으나 engagement가 낮아 전환 기여도 제한적. 모바일 최적화 필요성 대두.

**→ 액션**:
- 모바일 메인 페이지 로딩 속도 및 CTA 가시성 개선
- 모바일에서도 핵심 정보를 빠르게 파악할 수 있도록 구조 개선
- 모바일 사용자를 위한 간소화된 데모 요청 플로우 설계
