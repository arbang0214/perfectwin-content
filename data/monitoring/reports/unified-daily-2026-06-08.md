## 5️⃣ 오늘의 종합 인사이트

### 1. 홈페이지 트래픽 급증하지만 참여도 낮음으로 방치 탭 의심
- **현상**: 홈페이지 세션 16건(전일 대비 +1,500%), 하지만 참여율(10초+ 머물거나 2+ 페이지 본 세션 비율) 25%로 주말과 동일 수준. 평균 체류 시간도 7초로 매우 짧음.
- **왜**: Direct 트래픽이 11세션으로 주도하는데, 대부분 메인 페이지(/)에서 바로 이탈. 월요일 업무 시작으로 탭을 켜두기만 하고 실제로는 읽지 않는 방치 탭 패턴과 일치.
- **임팩트**: 실제 관심 있는 방문자보다 허수 트래픽이 많아 전환 가능성 낮음. 브랜드 인지도 상승으로 해석하기엔 이탈률 75%로 너무 높음.
- **→ 액션**: 메인 페이지의 첫인상을 강화하거나, 진짜 관심사를 파악할 수 있는 더 구체적인 CTA를 상단에 배치. 예: "SAP 테스트 자동화 무료 체크리스트 다운로드"

### 2. 검색 유입 품질 낮음 — 검색자 의도와 콘텐츠 미스매치
- **현상**: Google Organic 5세션이지만 참여율 20%, 평균 체류 3초로 매우 낮음. 페이지/세션 1.0으로 한 페이지 보고 즉시 이탈.
- **왜**: GSC에서 대부분 키워드가 둘째 페이지(11~20위) 이하에 위치하고, 표본 부족(노출 1~3건)으로 정확한 검색 의도 파악 어려움. 검색자가 원하는 정보와 실제 콘텐츠 간 갭 존재 추정.
- **임팩트**: 검색 트래픽은 늘어나지만 전환으로 이어지지 않는 구조. 검색엔진 최적화 효과가 양적 성장에 머물러 있음.
- **→ 액션**: 상위 랭킹 키워드("conversion testing strategy" 2위, "automation test script" 10위) 대상으로 메타 디스크립션 A/B 테스트. 검색 의도에 더 정확히 맞는 정보 제공 명시.

### 3. 블로그 CTA 전환 전무 — 콘텐츠는 읽지만 행동으로 안 이어짐
- **현상**: 영문 블로그 10방문, 한글 3방문이지만 CTA 클릭 0건. CVR(CTA 클릭 ÷ 방문) 0%로 B2B 정상 범위 1-3% 대비 매우 낮음.
- **왜**: 블로그 내부 이동(blog.perfectwin.ai 4건)이 30.8%로 높아 콘텐츠 자체엔 관심 있지만, 데모 신청이나 문의 같은 전환 행동으로 연결되지 않음. CTA 버튼 위치/문구 또는 전환 장벽 이슈.
- **임팩트**: 고품질 콘텐츠 읽는 잠재 고객을 리드로 전환하지 못해 기회 손실. 마케팅 ROI 저하.
- **→ 액션**: 인기 포스트("SAP S/4HANA Migration Testing") 내 CTA 배치 최적화. "무료 데모 신청" 대신 "SAP 마이그레이션 체크리스트 다운로드" 같은 낮은 진입장벽 오퍼 테스트.
