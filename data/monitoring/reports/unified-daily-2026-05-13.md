## 4️⃣ 오늘의 종합 인사이트

**1. 데모 신청자는 직접 의도적 접근 — 브랜드 인지도 신호**
- **현상**: 데모 신청 완료 1건이 모두 데모 페이지 직접 진입(direct)에서 발생. 홈페이지나 블로그를 거치지 않음.
- **왜**: 일반적인 콘텐츠 마케팅 퍼널(블로그→홈→데모)을 거치지 않고 바로 `/contact-us/request-demo`로 진입한 것은 PerfecTwin을 사전에 인지하고 있었다는 의미.
- **임팩트**: 고품질 리드. 브랜드 인지도가 형성되어 의도적으로 찾아온 잠재 고객으로 전환 가능성이 높음.
- **→ 액션**: 영업팀에 리드 품질이 높을 가능성을 공유. 외부 채널(LinkedIn 개인 계정, 오프라인 네트워킹, 다크 소셜)에서의 브랜드 노출 활동 지속 강화.

**2. 홈페이지 트래픽 2배 증가하지만 참여 품질은 소폭 하락**
- **현상**: 세션 43개(+115%), 참여율 39.53%(-5.47%p), 평균 체류시간 54.1초(-46.4초).
- **왜**: 트래픽 증가폭에 비해 참여율과 체류시간이 하락한 것은 저품질 유입 증가. 하지만 세션당 페이지뷰는 1.56(+0.26)으로 개선되어 방치 탭보다는 관심 있지만 빠르게 둘러본 방문자들.
- **임팩트**: 양적 성장은 있으나 질적으로는 혼재. 참여율 39.53%는 50% 기준 미달이지만 극도로 낮지는 않음.
- **→ 액션**: Direct 트래픽이 25세션으로 가장 많은데, 이 중 일부는 다크 소셜일 가능성. GA4에서 Direct 유입의 랜딩 페이지 분포 확인해서 메인 페이지 외 특정 페이지로 들어온 비율 분석.

**3. 블로그는 영문이 압도적이지만 자연 검색은 전무**
- **현상**: 영문 블로그 26방문, 한글 1방문. 영문 CTA 클릭 2건(CVR 7.7%), 한글 0건. 하지만 영문 organic 0, 한글 organic 1.
- **왜**: 영문 블로그는 Direct(15)와 inblog.ai(9) 의존도가 높아 자연 검색 효과가 없음. GSC 데이터에서 노출은 319건이지만 클릭 0건인 것과 일치.
- **임팩트**: 블로그 콘텐츠가 검색에서 발견되지 않고 있어 신규 유입 창출에 한계. 현재는 기존 방문자 재방문에 의존.
- **→ 액션**: GSC에서 노출 319건에도 클릭 0건인 원인 분석. 평균 순위 17.6은 첫 페이지 하단으로 메타 제목/설명 개선 검토. 특히 노출 1위 페이지인 SAP Cloud ALM(48 노출, 10.2순위) 우선 최적화.

**4. 한국 시장의 높은 참여도 — 타겟 시장 적합성**
- **현상**: 한국 세션 10개(23.3%), 참여율 60%로 미국(30.43%)보다 2배 높음.
- **왜**: 한국은 SAP 도입 기업이 많고 PerfecTwin의 메시징이 현지 시장 니즈와 잘 맞는 것으로 보임. 상대적으로 적은 트래픽이지만 고참여.
- **임팩트**: 한국 시장이 조기 adopter 가능성. 소수이지만 깊이 있게 사이트를 탐색하는 패턴.
- **→ 액션**: 한국 시장 대상 콘텐츠 마케팅 강화. 한글 블로그 방문이 1건뿐이므로 한글 SEO 콘텐츠 확대 및 한국 LinkedIn에서 영문 콘텐츠 공유 빈도 증가.

**5. 모바일 vs 데스크톱 참여도 차이 — UX 격차**
- **현상**: 데스크톱 참여율 42.31%, 모바일 35.29%. 하지만 모바일 체류시간(106초)이 데스크톱(20.1초)보다 5배 길음.
- **왜**: 데스크톱 체류시간이 매우 짧은 것은 빠른 스캔 후 판단하는 B2B 특성. 모바일 체류시간이 긴 것은 진짜 관심이 있어서 꼼꼼히 보는 것일 가능성.
- **임팩트**: 모바일 사용자가 더 깊이 있게 콘텐츠를 소비하지만 참여율은 낮다는 것은 모바일 CTA나 내비게이션에 문제 있을 수 있음.
- **→ 액션**: 모바일에서 데모 신청 버튼 위치, 크기, 문구 최적화. 모바일 사용자의 긴 체류시간을 전환으로 이어지게 하는 UX 개선.

## 📚 용어 정리

**데모 퍼널**
- 도달 — 데모 폼 페이지를 본 세션 수. "관심" 신호.
- 완료 — thank-you 페이지 도달 = 영업 메일 발생.
- 전환율 — 도달자 중 완료 비율. 정상 5~10%.
- 첫 진입 페이지 (Landing Page) — 사용자가 가장 먼저 본 페이지. "어떤 콘텐츠가 데려왔는가"의 가장 강한 신호.

**트래픽 (GA4)**
- 방문자 (Active Users) — 그날 들른 고유 사람 수.
- 세션 (Sessions) — 방문 횟수. 30분 비활성 후 새 세션.
- 페이지뷰 (Page Views) — 페이지 본 총 횟수.
- 참여율 (Engagement Rate) — 10초+ 머물거나 2+ 페이지 본 세션 비율. **50%+ 좋음**.
- 세션당 페이지뷰 — 한 방문당 평균 페이지 수. **1.5+ 탐색 의향**.
- 평균 체류 시간 — 한 세션 평균 사이트 머문 시간. **2분+ 좋음**.

**검색 (GSC / Bing)**
- 노출 (Impressions) — 검색 결과에 우리가 보인 횟수.
- 클릭 (Clicks) — 실제 클릭 수.
- CTR (Click-Through Rate) — 클릭/노출. 평균 2~3%, **5%+ 좋음**.
- 평균 순위 (Position) — 검색 결과 우리 평균 위치. **10 이하 = 첫 페이지**, 11~20 = 둘째 페이지(클릭 급락), 20+ = 사실상 미노출.

**블로그 (inblog)**
- visits — 블로그 방문 횟수.
- post — 포스트 페이지를 본 횟수 (홈/카테고리 제외).
- clicks (CTA 클릭) — 블로그 본문 안 버튼·링크 클릭 수. CTA = Call To Action.
- organic — 검색엔진에서 자연 유입된 방문.

**어트리뷰션**
- Source — 어디서 왔는지 (google, linkedin.com 등).
- Medium — 어떻게 왔는지 (organic=자연검색, social=소셜, referral=다른 사이트 링크, direct=직접 입력).
